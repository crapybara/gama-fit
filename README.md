## What is GammaFit

GammaFit is a lightweight self-care and fitness tracking platform focused on long-term progression, consistency, and user-controlled tracking systems.

The project is designed around:

- local-first workflows
- lightweight architecture
- low resource usage
- long-term progression tracking
- self-hosted deployment
- flexible user-created systems

## Philosophy

GammaFit is designed around a simple idea:

> Software should support long-term human progression instead of encouraging short-term engagement loops.

Core principles:

- Consistency
- Progression
- Local-first usability
- Data ownership
- Low system overhead
- Long-term sustainability

GammaFit is intended to function as a long-term progression and self-development platform rather than a temporary motivation tool.
